# Coffee Sales Dashboard ☕

## 📌 Project Overview
This project is an interactive Coffee Sales Dashboard built using Power BI.

The dashboard analyzes:
- Total Sales
- Total Orders
- Average Order Value
- Sales Trends
- Category Performance
- Product Performance
- Hourly Sales Insights

---

## 🛠 Tools Used
- Power BI
- Power Query
- DAX
- CSV Dataset

---

## 📊 Dashboard Features
- KPI Cards
- Sales Trend Analysis
- Category Split
- Top Products
- Hourly Sales Analysis
- Interactive Visual Design
